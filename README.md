# Interview Questions Repository

A community-driven collection of real interview experiences, questions, and preparation notes for internships and entry-level software roles.

The goal of this repository is simple: **help future candidates walk into interviews with a little more confidence.**

Instead of wondering what an interview might look like, students can learn from people who have already been through the process.

> **This repository is built on shared experiences, not memorized answers.** Use these contributions to prepare, understand expectations, and improve your own responses.

---

## 📚 Repository Structure

Each company has its own Markdown file containing:

* Interview details
* Questions that were asked
* Sample or personal answers (optional)
* Personal reflections
* Preparation tips
* Lessons learned

Example:

```text
Interview-Questions/
├── TEMPLATE.md
├── interviews/
     ├── Company-1.md
     ├── Company-2.md
     └── ...
```

## 🤝 Contributing

Contributions are always welcome!

If you've interviewed for an internship or an entry-level software role, consider sharing your experience.

To contribute:

1. Fork this repository.
2. Create a new Markdown file using `TEMPLATE.md`.
3. Name the file using the format `<CompanyName>_<InterviewType>.md` (for example, `Tkxel_AI_Screening_Interview.md`).
4. Fill in your interview experience.
5. Open a Pull Request.

That's it!

Even a single interview experience can help someone else prepare.

---

## 📌 Contribution Guidelines

Please:

* Share only questions that you personally encountered.
* Write from your own experience.
* Keep information accurate and respectful.
* Follow the provided template whenever possible.
* Feel free to improve formatting or fix mistakes in existing files through pull requests.

Please **do not**:

* Share confidential or proprietary information.
* Upload company assessment solutions that are not meant to be public.
* Include personal or sensitive information about interviewers or other candidates.
* Copy content from paid interview-preparation platforms.

---

## 💡 Why This Repository?

Preparing for interviews is difficult when you don't know what to expect.

By sharing interview experiences, we can build a resource that helps students:

* Understand interview patterns.
* Identify commonly asked questions.
* Learn from real experiences.
* Prepare more effectively.
* Reduce unnecessary anxiety before interviews.
